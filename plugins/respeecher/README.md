# Respeecher plugin for LiveKit Agents

This package contains the Respeecher plugin for voice synthesis with LiveKit Agents.
Refer to the [documentation](https://docs.livekit.io/agents/overview/) for information
on how to use it, or see the [repository](https://github.com/livekit/agents-js) for
more information about the framework as a whole.
