# Speechify plugin for LiveKit Agents

The Agents Framework is designed for building realtime, programmable
participants that run on servers. Use it to create conversational, multi-modal
voice agents that can see, hear, and understand.

This package contains the Speechify plugin, which provides streaming text-to-speech
with word-level timestamps. Refer to the [documentation](https://docs.livekit.io/agents/overview/)
for information on how to use it, or browse the [API
reference](https://docs.livekit.io/agents-js/modules/plugins_agents_plugin_speechify.html).
See the [repository](https://github.com/livekit/agents-js) for more information
about the framework as a whole.

This plugin is maintained by Speechify. Issues and contributions specific to the
plugin are welcome at
[Speechify-AI/livekit-plugin-js](https://github.com/Speechify-AI/livekit-plugin-js).
